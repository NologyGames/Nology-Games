# Nology Games Installers

You will find here all the Windows installers for Nology Games developed in 2000's. 

## Instructions

1. Download the game.
2. Extract the ZIP file contents.
3. Execute the Setup.exe file inside it.
4. Follow the installer instructions.
5. Run the game through the Start Menu->Nology Games->"Name of the Game"->"Name of the Game" (see below):



## How to uninstall
1. Go to the Start Menu->Nology Games->"Name of the Game" and execute Uninstall.
or
2. Go to Start Menu->Settings->Control Panel. Find the game in the list and uninstall it.
