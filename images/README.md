# Images — save before deleting the source site

The site uses **local** image paths. You must save these files from the live Google site **before** you delete it.

## Artwork (5 images)

Save each image from the [Artwork page](https://sites.google.com/site/nologygames/artwork) into this folder with these exact names:

| Save as        | Source |
|----------------|--------|
| `artwork-1.jpg` | First image on Artwork page |
| `artwork-2.gif` | Second image |
| `artwork-3.gif` | Third image |
| `artwork-4.jpg` | Fourth image |
| `artwork-5.jpg` | Fifth image |

**How to save:** Right‑click each image → “Save image as…” → choose this folder and the filename above.
