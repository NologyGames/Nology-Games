Download the full version of each game below, then follow How to install.

  • Koko Arena (Nology-KokoArenaSetup.exe)
  • JoKenPo (Nology-JoKenPoSetup.exe)
  • Cookie Chef (Nology-CookieChefSetup.exe)
  • Hello! (Nology-HelloSetup.exe)
  • Super Popcorn Machine (Nology-SPMSetup.exe)
  • Chip$ (Nology-ChipSSetup.exe)

If the files are not yet available here, you can also get them from the legacy Google Drive folder.

How to install

  1. Download the game from the list above.
  2. Extract the ZIP file contents.
  3. Execute the Setup.exe file inside it.
  4. Follow the installer instructions.
  5. Run the game through the Start Menu → Nology Games → "Name of the Game" → "Name of the Game" (see below).

How to uninstall

  1. Go to the Start Menu → Nology Games → "Name of the Game" and execute Uninstall.

or

  1. Go to Start Menu → Settings → Control Panel. Find the game in the list and uninstall it.
